## 📚 Basic Git Commands

### Check Git Status

```bash
git status
```

### Add Files

```bash
git add .
```

### Commit Changes

```bash
git commit -m "Your commit message"
```

### Push Changes

```bash
git push origin main
```

### Pull Latest Changes

```bash
git pull origin main
```

### Create a New Branch

```bash
git checkout -b feature-branch
```

### Switch Branch

```bash
git checkout main
```

### Merge Branch

```bash
git merge feature-branch
```
